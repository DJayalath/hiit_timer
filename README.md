# hiit_timer

A timer for high intensity interval training (HIIT)

## Features

### Settings
- Set rep time
- Set cycles in set
- Set number of sets
- Set break interval between sets

### Timer
- Rep timer
- Rest timer
- Cycle tracker
- Set tracker
- Wakelock
- Orientation lock
- Sound tones
